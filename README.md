Colourball is a 2D, top-down, puzzle maze game where players control a small ball that rolls in different coloured paint to change its properties. Different colours allow the players to defeat enemies and destroy matching coloured walls. Players need to strategically plan out what colour they are to reach the end flag. This unique retro game will feature colour-navigating puzzles, destructible environments, maze navigation and very light combat.

Colourball is designed to be easy to learn and suitable for all ages. One of the biggest selling points of the game is its educational use, as it can be used to teach kids about colour theory. The game is also fairly simple to understand, as the core mechanic is changing colours and matching the ball to the colour of the next wall we need to break. Through experimentation with colour mixing and strategic puzzle solving, the player will gradually gain control over the environment and eventually reach the end of the level.
Our target audience is casual players, retro game enjoyers, and younger players who are interested in puzzle games, but also education games that focus on colour theory.

Controls: D-Pad for movement, start for starting in main menu, a/b to attack enemies or nah
