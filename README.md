Colourball is a 2D, top down, puzzle game where players control a small ball that rolls in different coloured paint to 
change its properties. Different colours allow the players to defeat enemies and destroy matching coloured walls. 
Players need to strategically plan out what colour they are to reach the flag. This unique retro game will feature 
colour mixing puzzles, destructible environments, maze navigation and very light combat.


Controls: D-Pad for movement, start for starting in main menu, a/b to attack enemies or nah 

Use with the Mesen emulator. 

All backend files are included in GitHub, such as the scripts and graphic assets. You can download the NES ROM folder, unzip the file called 
ColourBall, then drag and drop this onto your Mesen emulator to play.
